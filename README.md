## ✍️ 커밋 & 파일 규칙

### 파일명 규칙
```
플랫폼_레벨_문제명.java

예시)
PGS_lv1_두개뽑아서더하기.java
PGS_lv2_타겟넘버.java
BOJ_silver_에라토스테네스의체.java
```

### 커밋 메시지 규칙
```
[플랫폼/레벨] 문제명 - 풀이방식

예시)
[PGS/Lv1] 두 개 뽑아서 더하기 - HashSet 활용
[PGS/Lv2] 타겟 넘버 - DFS 재귀
[BOJ/Silver] 에라토스테네스의 체 - 소수 판별
```

### 풀이 파일 내 주석 규칙
```java
/**
 * 문제: 문제 이름 (플랫폼 Lv.X)
 * 링크: https://...
 * 접근: 풀이 방식 한 줄 요약
 * 시간복잡도: O(n)
 *
 * 풀이 메모:
 * - 처음 시도한 방법과 막힌 부분
 * - 최종 해결 방법
 */
```
