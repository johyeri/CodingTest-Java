## ✍️ 커밋 & 파일 규칙

### 파일명 규칙
```
platform/level/id. title.java

예시)
프로그래머스/lv2/12345. 타겟 넘버
백준/Gold/1000. A＋B
```

### 커밋 메시지 규칙
```
[레벨] 문제명, 시간, 메모리 

예시)
[level 2] Title: 다리를 지나는 트럭, Time: 21.99 ms, Memory: 82.3 MB
```

### 풀이 파일 내 주석 규칙
```java
/**
 * 접근: 풀이 방식 한 줄 요약

 * - 처음 시도한 방법과 막힌 부분
 * - 최종 해결 방법
 */
```
